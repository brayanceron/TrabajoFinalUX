Abstract: 
Se pretende realizar un software (de escritorio o bien 'on line') donde a partir de una URL se pueda: - obtener un mapa de navegación del sitio detectando enlaces, formularios y otros elementos estructurales. - probar el sitio web para verificar su buen funcionamiento: respuesta, enlaces rotos, formularios enviados, etc. - lanzar pruebas de carga (con múltiples accesos) con el fin de probar la robustez y tolerancia del sitio.
